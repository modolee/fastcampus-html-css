# `background-repeat`

배경 이미지의 반복을 설정 <kbd>`개별`</kbd>

| 값        | 의미                              | 기본값 |
| --------- | --------------------------------- | ------ |
| repeat    | 배경 이미지를 수직, 수평으로 반복 | repeat |
| repeat-x  | 배경 이미지를 수평으로 반복       |        |
| repeat-y  | 배경 이미지를 수직으로 반복       |        |
| no-repeat | 반복 없음                         |        |
