# `background-position`

요소가 스크롤될 때 배경 이미지의 스크롤 여부(특성) 설정 <kbd>`개별`</kbd>

| 값     | 의미                                                                   | 기본값 |
| ------ | ---------------------------------------------------------------------- | ------ |
| scroll | 배경 이미지가 요소를 따라서 같이 스크롤 됨                             | scroll |
| fixed  | 배경 이미자가 뷰포트(viewport)에 고정되어, 요소와 같이 스크롤되지 않음 |        |
| local  | 요소 내 스크롤 시 배경 이미지가 같이 스크롤 됨                         |        |
