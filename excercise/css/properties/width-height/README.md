# 속성

## `width`

| 값   | 의미                            | 기본값 |
| ---- | ------------------------------- | ------ |
| 단위 | px, em, rem, %, vw 등 단위 지정 | auto   |
| auto | 브라우저가 너비를 계산          |        |

- block 요소 : auto = 100 %
- inline 요소 : auto = 0 **(지정 불가)**

## `height`

| 값   | 의미                             | 기본값 |
| ---- | -------------------------------- | ------ |
| 단위 | px, em, rem, %, vh, 등 단위 지정 | auto   |
| auto | 브라우저가 높이를 계산           |        |

- block 요소 : auto = 0
- inline 요소 : auto = 0 **(지정 불가)**
