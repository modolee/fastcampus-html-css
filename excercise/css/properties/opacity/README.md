# `opacity`

요소의 투명도를 지정

| 값   | 의미                          | 기본값 |
| ---- | ----------------------------- | ------ |
| 숫자 | `0`부터 `1`사이의 소수점 숫자 | 1      |

## 사용법

```css
/* opacity: 투명도; */

.half {
  opacity: 0.5; /* 50% 투명도, 반투명 */
}

.transparent {
  opacity: 0; /* 0% 투명도, 투명 */
}

.box1 {
  opacity: 0.75; /* 75% 투명도, 반투명 */
}

.box2 {
  opacity: 1; /* 100% 투명도, 불투명 */
}
```

## 주의사항

- `display: none`과 `opacity: 0`은 동일하지 않음
- `display: none`: 요소 자체가 존재하지 않음
- `opacity: 0`: 요소는 존재하지만, 보이지만 않음
