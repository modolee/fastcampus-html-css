# `text-decoration`

문자의 장식(line)을 설정

| 값             | 의미                       | 기본값 |
| -------------- | -------------------------- | ------ |
| `none`         | 선 없음                    | `none` |
| `underline`    | 밑줄을 지정                |        |
| `overline`     | 윗중을 지정                |        |
| `line-through` | 중앙 선(가로지르는)을 지정 |        |
