# `text-align`

문자 정렬 방식을 지정

| 값        | 의미                                                     | 기본값 |
| --------- | -------------------------------------------------------- | ------ |
| `left`    | 왼쪽 정렬                                                |        |
| `right`   | 오른쪽 정렬                                              |        |
| `center`  | 가운데 정렬                                              |        |
| `justify` | 양쪽 맞춤<br>(화면에 2줄 이상 표시 될 때, 강제 줄바꿈 X) |        |
