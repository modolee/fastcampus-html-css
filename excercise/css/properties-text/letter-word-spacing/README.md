# `letter-spacing`

문자의 자간(글자 사이 간격)을 설정

| 값       | 의미                            | 기본값   |
| -------- | ------------------------------- | -------- |
| 단위     | px, em, rem, %, vw 등 단위 지정 |          |
| `normal` | 문자 사이의 일반 간격           | `normal` |

# `word-spacing`

단어 사이(띄어쓰기)의 간격을 설정

| 값       | 의미                            | 기본값   |
| -------- | ------------------------------- | -------- |
| 단위     | px, em, rem, %, vw 등 단위 지정 |          |
| `normal` | 단어 사이의 일반 간격           | `normal` |
