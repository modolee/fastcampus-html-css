# flex-grow

Item의 증가 너비 비율을 설정

- 숫자가 크면 더 많은 너비를 가짐
- Item이 `가변 너비`가 아니거나, 값이 0일 경우 효과가 없음

| 값   | 의미                         | 기본값 |
| ---- | ---------------------------- | ------ |
| 숫자 | Item의 증가 너비 비율을 설정 | 0      |

## 사용법

```css
/* flex-grow: 증가너비; */

/* 25% 너비 */
.item1 {
  flex-grow: 1;
}

/* 50% 너비 */
.item2 {
  flex-grow: 2;
}

/* 25% 너비 */
.item3 {
  flex-grow: 1;
}
```
