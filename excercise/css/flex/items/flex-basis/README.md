# flex-basis

Item의 (공간 배분 전) 기본 너비를 설정

- 값이 `auto`일 경우, `width, height` 등의 속성으로 Item의 너비를 설정할 수 있음
- 하지만 단위 값이 주어질 경우 설정할 수 없음

| 값   | 의미                             | 기본값 |
| ---- | -------------------------------- | ------ |
| auto | 가변 Item과 같은 너비            | auto   |
| 단위 | px, em, rem, %, vh, 등 단위 지정 |        |

## 사용법

```css
/* flex-basis: 기본너비; */
```
