# grid-row

- grid-row-start와 grid-row-end의 단축 속성

> 각 속성을 `/`로 구분

```css
/* .item {
  grid-row: <grid-row-start> / <grid-row-end>;
} */

.item {
  grid-row-start: 1;
  grid-row-end: 2;
}

.item {
  grid-row: 1 / 2;
}
```

# grid-column

- grid-column-start와 grid-column-end의 단축 속성

> 각 속성을 `/`로 구분

```css
/* .item {
  grid-column: <grid-column-start> / <grid-column-end>;
} */

.item {
  grid-column-start: -1;
  grid-column-end: -3;
}

.item {
  grid-column: -1 / -3;
}
```
