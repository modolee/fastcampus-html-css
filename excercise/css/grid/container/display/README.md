# display

- Grid Container를 정의
- 정의 된 컨테이너의 자식 요소들은 자동으로 Grid Items로 정의

> 그리드를 사용하기 위해 컨테이너에 필수로 작성

| 값          | 의미                                | 기본값 |
| ----------- | ----------------------------------- | ------ |
| grid        | Block 특성의 Grid Container를 정의  |        |
| inline-grid | Inline 특성의 Grid Container를 정의 |        |

```css
.container {
  display: grid;
}
```
